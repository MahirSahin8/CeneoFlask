CeneoScraper

Scrape, store, browse and analyse product opinions from Ceneo.pl — Poland's price-comparison marketplace.

Start extracting View products
What it does

Enter a Ceneo product code and the app downloads every opinion across all pages, extracting authors, scores, recommendations, pros, cons, vote counts and dates.

Purpose

Built as a university project to demonstrate web scraping, data modelling, persistence and visualisation. Opinions are saved as JSON and can be exported to CSV or XLSX.

Analyse

Browse opinions in a sortable, filterable table and view charts of the recommendation split and the distribution of star ratings.

About & libraries used

CeneoScraper is built with the following open-source libraries:

  • Flask — web framework and routing
  • requests — HTTP client for fetching opinion pages
  • BeautifulSoup4 — HTML parsing and data extraction
  • pandas + openpyxl — CSV / XLSX export
  • Bootstrap 5 — responsive UI styling
  • DataTables — sortable / filterable opinion table
  • Chart.js — recommendation and rating charts
